# Brand Assets

Drop the client's brand assets here BEFORE designing:

- `logo.svg` or `logo.png` — the client's logo
- `colors.md` — primary, secondary, accent hex values
- `fonts.md` — heading + body font choices (Google Fonts names work)
- `photos/` — any real photography the client provides
- `style-notes.md` — anything else (mood board notes, "no stock photos", etc.)

If a logo is here, the frontend uses it. If hex colors are defined, those are used — never invented.

If this folder is empty, the design falls back to trade-themed placeholders from `src/placeholders.ts`.
